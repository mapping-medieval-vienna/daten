# DB5457

## Hausbiographie

Kiemarkt  
durchgehend ein halbes Haus  
mitsamt einem Stocklein im Berghof, ab 1441 stößt das halbe Haus auch in den Berghof  

Peter **Alderman**  
<a href="#GB-C-114-4"><code>GB-C, 114-4</code></a>  1421 Hanns **Alderman** -- (keine Nachbarn)  
Hanns heiratet Clara  
Clara heiratet Jorg von **Ernstprun**  
<a href="#GB-D-0364-1"><code>GB-D, 0364-1</code></a> 1441 Jorg -- N1  
<a href="#GB-D-0596-2"><code>GB-D, 0596-2</code></a> 1468 Jorgs Kinder -- N1
